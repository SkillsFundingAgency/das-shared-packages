# SFA.DAS.Validation

Automatic model validation for MVC applications that are using the OMIOMO (One Model In, One Model Out) & PRG (Post/Redirect/Get) patterns. 

## Configuration

### MVC Core

```c#
services.AddMvc(o => o.AddValidation());
```
